# C Language Practice

C 언어 기본 문법을 실습하고 정리한 저장소입니다.

## 📂 폴더 구조

- `01_variables_constants`: 변수, 상수 실습
- `02_io`: 입출력 실습
- `03_operators`: 연산자 실습
- `04_conditionals`: 조건문 실습 (if문, switch문)
- `05_loops`: 반복문 실습 (for, while, 구구단, 무한 루프 등)

## ✨ 실습 환경

- Visual Studio Code  
- Replit / PowerShell / GitHub
